кратко обо мне

Привет! Меня зовут Дина, мне 20 лет. Учусь в ИТМО на ИТИПе на 3 курсе, работаю в Научном Центре Когнитивных Разработок продакт-менеджером, руковожу студенческим клубом конфликтологии ITMO Negotiations.
В свободное время стараюсь изучать что-то новое по работе, вижусь с друзьями, играю на фортепиано, фотографирую на пленку и бла бла бла.

Кусок кода из моей лабораторной работы по ООП, класс, описывающий сущность. 

                
public class IEntity {

    int Id = 0;

    public int GetId() {
        return Id;
    }
}
Суть лабораторной заключалась в том, чтобы реализовать механизм автоматизации отчетов о работе в команде. Есть несколько сущностей - сотрудник, задача и отчет. Отчеты бывают за день и за спринт. В работе необходимо использовать принципы ООП.
